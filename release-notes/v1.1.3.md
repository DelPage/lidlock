# LidLock 1.1.3

LidLock now stays available in the macOS menu bar after its main window closes.

## Download

The attached `LidLock.dmg` is signed with Apple Developer ID and notarized by
Apple.

SHA-256:

```text
fe735f2a9cebdbd5807163d4099c75df6141d70ca36aa91265562caf1d78e8af  LidLock.dmg
```

## What Changed

- Closing the main window leaves LidLock running in the menu bar.
- **Open LidLock** brings the window back from the menu bar.
- **Quit LidLock** remains the explicit way to exit and run quit-time cleanup.
- Hovering over the menu-bar icon shows the current sleep-protection status.

## Requirements

- macOS 13 Ventura or newer.
- A MacBook for lid-close workflows. Other Macs can still use keep-awake and
  display controls.

## Privacy

LidLock has no accounts, analytics, telemetry, advertising, or automatic network
access during normal operation. The optional helper is local to your Mac and is
used only for LidLock's lid-close sleep setting.

Privacy policy: https://delpageinc.com/privacy/lidlock
