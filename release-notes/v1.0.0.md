# LidLock 1.0.0

Initial public release of LidLock for macOS.

## Download

The attached `LidLock.dmg` is signed with Apple Developer ID and notarized by Apple.

SHA-256:

```text
1bd59b93a726cca08aff3bf222fb70d9e88d5c680ff474e07e20714f23923799  LidLock.dmg
```

## Requirements

- macOS 13 Ventura or newer.
- A MacBook for lid-close workflows. Other Macs can still use keep-awake and display controls.

## Privacy

LidLock has no accounts, analytics, telemetry, advertising, or automatic network access during normal operation.

Privacy policy: https://delpageinc.com/privacy/lidlock
