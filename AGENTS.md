# LidLock Public Repo Notes

This repository is public and intentionally closed-source.

Do not add the LidLock application source tree, signing material, provisioning profiles, private keys, `.env` files, build intermediates, notarization credentials, or local DerivedData here.

Allowed content:

- README and documentation.
- Screenshots, GIFs, icons, and product media.
- Issue templates.
- Public release notes and checksums.
- Signed and notarized public release artifacts attached through GitHub Releases.
